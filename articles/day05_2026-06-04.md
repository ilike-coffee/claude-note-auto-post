# Claudeに「もっとよく」答えてもらう3つのコツ

こんにちは！Claude素人です。

Claudeを使い始めてみたものの、「なんかイマイチな答えが返ってくることがある…」と感じたことはありませんか？

私も最初はそうでした。同じClaudeを使っているのに、ちょっとした「聞き方の違い」だけで、返ってくる答えのクオリティが全然違う。これって最初は不思議に感じるんですよね。

でも実は、Claudeに限らずAIへの「お願いの仕方」にはコツがあります。今日はその中でも特に効果的な**3つのコツ**を、Before/Afterの具体例を交えながらご紹介します。

---

## コツ①：「具体的に」お願いする

一番シンプルで一番大事なコツです。

AIに曖昧な指示を出すと、AIも曖昧な答えを返してきます。「ふわっとした質問にはふわっとした答え」が返ってくる、というイメージですね。

### Before（曖昧な指示）

```
メールを書いて
```

→ Claudeはどんなメールかわからないので、汎用的すぎる文章を作ってきます。

### After（具体的な指示）

```
取引先の田中さんへ、先日の打ち合わせのお礼と
来週月曜日の14時にWeb会議を設定したいという内容の
ビジネスメールを書いてください。
簡潔に、200字以内でお願いします。
```

→ ぐっと使える内容になります！

**ポイントは「5W1H」を意識すること。**

- **Who（誰に）**：田中さんへ
- **What（何を）**：お礼＋会議設定の依頼
- **When（いつ）**：来週月曜14時
- **How（どうやって）**：200字以内・ビジネス文体

全部を毎回入れる必要はありませんが、「この情報があればClaudeも判断できるな」という情報を添えてあげると、ぐっと精度が上がります。

---

## コツ②：「役割（キャラ）」を与える

これ、やってみると効果の大きさに驚くコツです。

「あなたは〇〇の専門家です」と最初に宣言するだけで、Claudeの回答トーンや視点がガラッと変わります。

### Before（役割なし）

```
ダイエットについて教えてください
```

→ 一般的な情報が返ってきます。浅く広い内容になりがちです。

### After（役割あり）

```
あなたはスポーツ栄養士の専門家です。
30代の運動習慣がない会社員が、
無理なく3ヶ月で5kg痩せるための食事アドバイスをしてください。
```

→ 専門的な観点から、その人に合った具体的なアドバイスが返ってきます。

「役割を与える」は英語で **Role Prompting（ロールプロンプティング）** と呼ばれる、広く使われているテクニックです。

他にも、こんな役割設定が使えます。

- 「あなたは経験豊富な編集者です。この文章を読みやすく添削してください」
- 「あなたは小学生でも分かるように説明するのが得意な先生です」
- 「あなたは厳しいフィードバックをくれるビジネスコーチです」

役割によって、答えの「深さ」も「トーン」も変わってくるので、目的に合った役割を選んでみてください。

## コツ③：「例を示す」（Few-shotプロンプティング）

「こんな感じで答えてほしい」という**お手本を見せる**テクニックです。

百聞は一見に如かず、という言葉がありますが、AIにとっても同じ。「こういう形式で答えてほしい」と説明するより、「例を1つ見せる」ほうが、はるかに正確に意図が伝わります。

### Before（例なし）

```
この商品の紹介文を書いてください
商品名：保温マグカップ
特徴：6時間保温・軽量・おしゃれなデザイン
```

→ 普通の紹介文は書いてくれますが、自分のnoteや販促資料に使うにはもう一歩…というケースも。

### After（例あり）

```
以下の例のような文体・構成で、商品紹介文を書いてください。

【例】
「どんな季節でも、好きな温度でコーヒーを楽しみたい方へ。
このマグカップは、12時間保温・保冷対応。
朝入れたコーヒーが、昼過ぎでもあったか。」

【今回書いてほしいもの】
商品名：保温マグカップ
特徴：6時間保温・軽量・おしゃれなデザイン
```

→ 例に近いトーンと構成で書いてくれます！

この「例を見せる」やり方は、**Few-shot（フューショット）プロンプティング**と呼ばれています。「few shot = 少ない例を見せる」という意味です。

自分のブログの文体を真似させたい、メールのフォーマットを揃えたい、といったときにとても役立ちます。

---

## 3つのコツまとめ

| コツ | ポイント |
|------|----------|
| ①具体的に | 5W1Hを意識して情報を盛り込む |
| ②役割を与える | 「あなたは〇〇の専門家です」と宣言する |
| ③例を示す | 「こんな感じで」とお手本を見せる |

これら3つは、組み合わせて使うとさらに効果的です。

たとえば――

```
あなたは人気note作家です（役割）。
以下の構成で記事の導入文を書いてください（具体的に）。

【例】
「毎朝、コーヒーを飲みながら10分だけ読書する。
そんな習慣を始めて3ヶ月で、仕事の成果が変わった話をします。」（例を示す）

今回のテーマ：「毎日5分の英語学習で起きた変化」
```

こんな風に3つを掛け合わせると、かなりクオリティの高い結果が返ってきます。

## 最後に

「プロンプトを工夫する」というと難しそうに聞こえますが、要は**「相手（AI）に伝わりやすい言い方をする」**だけのことです。

人間に仕事をお願いするとき、「なんかいい感じにやって」より「〇〇さんに、こういう内容で、これくらいの量でお願い」と頼むほうが、ちゃんとやってもらえますよね。Claudeも同じです。

今日紹介した3つのコツ、ぜひ明日から試してみてください。最初は少し面倒かもしれませんが、すぐに「あ、これだ！」という感覚が掴めてくるはずです。

次回は、**「Claudeにキャラを与えると何が変わるか」**を実験してレポートします。今日の「役割を与える」コツをもっと深掘りする内容です。お楽しみに！
